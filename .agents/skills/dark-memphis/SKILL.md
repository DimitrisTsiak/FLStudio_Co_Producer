# Dark Memphis Beats Skill

This skill ensures that the generated midi files fit the style of dark memphis beats

## Instructions
0. **Set the tempo to 140-160 bpm**:
- All the midi files should have the same tempo

1. **Create the hi hat patterns**:
- Create hi hat patterns that fits the style of dark memphis beats
- The hi hat patterns should be 8th notes with some hi hat rolls using 16th and 32nd notes. Not to much but enough to be noticeable. all the midi notes should be on the C4 note, the rolls could be on different notes but not more than one octave apart from the C4 note.
- The hi hat one the beat should have a higher velocity that the off beat
- There should be at least 2 different hi hat patterns
- Each hi hat pattern should be 8 bar long
- Each hi hat pattern should be saved as a separate midi file

2.**Create the kick pattern**
- Create kick patterns that fits the style of dark memphis beats
- The kick pattern should give the bounce of the beat. It should always hit on the first beat of every two bars. The rest of the hits should be at the 4th and 6th beats but not always. you can also add kicks on the offbeat to complement the main pattern.
- Each kick pattern should be 8 bar long
- only use the C4 note
- avoid using notes that sit on 2nd and 4th 16th notes of each beat


3.**Create the snare pattern**
- Create snare patterns that fits the style of dark memphis beats
- The snare should always hit on the 3rd beat of each bar, you can add 1 or 2 snares at ofbeats but it's not nessecary. 
-  all the midi notes should be on the C4 note
- create a seccond pattern that only contains a snare fill at the end of the 8 bars.


4.**Create the low piano pattern**
- create a low piano midi file that fits the style of dark memphis beats.
- it should be 8 bars long.
- it shouldnt use many notes maximum three.
- the notes should be spaced out.
- the notes should be low in the piano range.
- prefer to use notes that are one semitone apart.
- the low piano should be in the Phrygian mode. The first note should be the tonic and it should be the most common note in the pattern.


5.**Create the high piano pattern**
- create a high piano midi file that fits the style of dark memphis beats.
- it should be 8 bars long.
- it should be in the minor scale that corresponds to the root note of the low piano.
- it should be repetitive and memorable.
- the notes should be high in the piano range. 
- it should make use of dark half step intervals but dont overdo it.

6. **Create the bass pattern**
- create a bass midi file that fits the style of dark memphis beats.
- it should be 8 bars long.
- it should be repetitive and memorable.
- they should follow the progression of the low piano.
- the should follow the kick pattern
- the main root note should be in the c4-c5 range
- you can add filler notes to make it more interesting in the second 4 bars. if you choose to do it don't overdo it. these notes could be one octave higher
